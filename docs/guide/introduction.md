# Introduction

## APIs


### Receive money
`POST` `/api/v1/trades`

### Query a receive money transaction
`GET` `/api/v1/trades/:trade`

### Send money
`POST` `/api/v1/transfers`

### Query a send money transaction
`GET` `/api/v1/transfers/:transfer`

### DaYangPay Balance

`GET` `/api/v1/user/balances`
