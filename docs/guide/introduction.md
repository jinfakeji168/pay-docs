# Introduction

## APIs

### Receive money

`POST` `/api/v1/trades`

### Get a transaction for receive money

`GET` `/api/v1/trades/:trade`

### Send money

`POST` `/api/v1/transfers`

### Get a transaction for send money

`GET` `/api/v1/transfers/:transfer`

### DaYangPay Balance

`GET` `/api/v1/user/balances`
